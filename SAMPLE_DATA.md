## What Makes a Good Test?

A good test should show:
1. ✅ AI correctly identifies matching skills
2. ✅ AI finds missing but relevant keywords
3. ✅ Feedback is specific and actionable
4. ✅ Match score is reasonable (not 0% or 100%)
5. ✅ Recommendations are practical

If results seem off, try:
- Simplifying the text (remove special characters)
- Making resume/JD more detailed
- Running analysis again (AI responses vary slightly)
- Checking that API key is working
